## 🚀 Getting Started

If you like this setup, feel free to **fork this repository** and customize it for your own use! You can change the text in `index.html`, update the styles in `style.css`, and make it your own.

## 🛠️ Deployment Instructions

To host this on your own GitHub Pages:
1. Push the code to your repository.
2. Go to **Settings > Pages**.
3. Under **Build and deployment > Source**, ensure **GitHub Actions** is selected.
4. The deployment will trigger automatically.
