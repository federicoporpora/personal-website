# Federico Porpora | Computer Engineering Portfolio

A personal portfolio website showcasing academic background, technical projects, and extracurricular leadership.

## Technologies
- **HTML5 / CSS3**: Responsive implementation using CSS variables, Flexbox, and Grid.
- **JavaScript**: Selective functionality for theme management and scroll-reveal interactions.
- **Design System**: Modern "Bento Box" layout with optimized light and dark modes.

## Site Structure
- **Home**: Professional profile and core methodology.
- **About**: Detailed academic background and technical proficiencies.
- **Projects**: Selection of key software engineering and automation projects.
- **Extracurriculars**: Showcase of leadership roles, athletics, and community involvement.

## Project Organization
- `index.html`, `about.html`, `projects.html`, `hobbies.html`: Core page implementations.
- `style.css`: Centralized styling and design tokens.
- `script.js`: Unified interaction logic.
- `assets/`: Optimized media and profile resources.
- `.gitignore`: Repository configuration for excluding sensitive or non-public data.
